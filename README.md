# Grace Upton - Landing Page

A beautiful, elegant landing page for Grace Upton designed to showcase content and direct visitors to the FanVue profile.

## Features

- Elegant title with gradient effects
- Interactive image carousel with 9 photos
- Auto-advancing carousel (4-second intervals)
- Fully clickable elements that link to FanVue
- Responsive design for all devices
- Smooth animations and transitions
- Modern gradient background

## Live Site

This site is hosted on GitHub Pages and can be accessed at:
`https://[username].github.io/grace-upton/`

## FanVue Profile

Visit Grace Upton's exclusive content at: [https://www.fanvue.com/graceupton/fv-1](https://www.fanvue.com/graceupton/fv-1)

## Structure

- `index.html` - Main landing page
- `style.css` - Styling and responsive design
- `script.js` - Carousel functionality and interactions
- `images/` - Photo gallery (images 1-9)

## Deployment

This site is configured for GitHub Pages deployment:

1. Push all changes to the `main` branch
2. Go to repository Settings > Pages
3. Set source to "Deploy from a branch"
4. Select `main` branch and `/ (root)` folder
5. Save and wait for deployment

## Local Development

Simply open `index.html` in a web browser to view the site locally.

## Technologies

- HTML5
- CSS3 (with gradients, transitions, and responsive design)
- Vanilla JavaScript
